# 🎯 Шашки Онлайн на C++

Многопользовательская игра в шашки с веб-интерфейсом на C++ сервере.

## 🚀 Возможности

- ✅ Игра по сети с разных устройств
- ✅ Полные правила шашек (дамки, взятие)
- ✅ Красивый веб-интерфейс
- ✅ Автообновление состояния
- ✅ Поддержка мобильных устройств

## 📦 Установка и запуск

### Linux/Mac:
```bash
mkdir build && cd build
cmake ..
make
./../bin/checkers_server
